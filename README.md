# Introdução/Requerimento
Cada grupo montará um cenário (empresa), onde deverá conter: Breve descrição da empresa, planta baixa dos prédios, quais os equipamentos cada área deverá trabalhar (levar em conta todo
o aspecto de hardware e os softwares que foram colocados no cenário), quais são os sistemas operacionais que os equipamentos utilizarão, topologia utilizada, esquema de endereçamento de IP, equipamentos de IoT que a empresa poderá utilizar e planos de gestão e governaça utilizados

# Especifiações e Software Utilizados
* Packet Tracer
* Figma
* HTLM e CSS
* Mooble

# Sales Pitch
Nosso grupo teve a idéia de criar um software que atua na área da saúde digital (HealthTech), com foco em engenharia de software aplicada a sistemas hospitalares. Sua missão é integrar aplicações móveis e sistemas web hospitalares, permitindo comunicação em tempo real entre paciente, servidor e médico.
